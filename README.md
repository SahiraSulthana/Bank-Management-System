<h1>Bank Management System</h1>
<p>A <b>Bank Management System</b> built using <b>Java and MySQL</b> that allows users to perform basic banking operations through a console-based menu. This project demonstrates <b>CRUD operations, JDBC connectivity, and core Java concepts</b>.</p>
<h2>Features</h2>
<ul>
  <li>Create a new bank account</li>
  <li>Deposit money</li>
  <li>Withdraw money</li>
  <li>Check account balance</li>
  <li>View account details</li>
  <li>Delete Account</li>
  <li>Secure database connectivity using MySQL</li>
</ul>
<h2>Technologies Used</h2>
<ul>
  <li><b>Java</b> (Core Java, JDBC)</li>
  <li><b>MySQL </b>(Database)</li>
  <li><b>JDBC Driver</b> for database connection</li>
  <li><b>Eclipse</b> (IDE)</li>
</ul>
<h2>Future Enchancements</h2>
<ul>
  <li>Add login authentication</li>
  <li>GUI using Swing / JavaFX</li>
  <li>Transaction history</li>
  <li>Online banking features</li>
</ul>
